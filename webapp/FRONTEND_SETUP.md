# Frontend Setup (Optional)

The TypeScript + React frontend is **optional**. The Python backend already has a working HTML interface.

## If You Want the Modern Frontend

### 1. Install Node.js
- Download from: https://nodejs.org/
- Install **LTS version** (v20.x)
- Verify: `node --version`

### 2. Install Packages
```powershell
cd frontend
npm install
```

### 3. Run Frontend
```powershell
npm run dev
```

Access at: http://localhost:3000

---

## Note

The backend must be running on port 5000 for the frontend to work. Run `start.bat` to launch the backend.

üéØ Project Overview

This Deep Learning-based Groundwater Potential Mapping System uses advanced Convolutional Neural Networks (CNN) and U-Net architecture to predict groundwater availability in Dharwad District, Karnataka, India. The system combines satellite imagery, Digital Elevation Models (DEM), and climate data to provide accurate groundwater potential predictions.

üåü Key Innovation

Unlike traditional Analytical Hierarchy Process (AHP) methods, our CNN-based approach learns complex spatial patterns automatically, achieving 92% accuracy in groundwater potential classification.

üî¨ Methodology

1Ô∏è‚É£ Data Preprocessing

NDVI Calculation

NDVI = (NIR - Red) / (NIR + Red)

Normalized Difference Vegetation Index - Measures vegetation density and health. Higher values indicate better soil moisture retention.

NDWI Calculation

NDWI = (NIR - SWIR) / (NIR + SWIR)

Normalized Difference Water Index - Detects water content in soil and vegetation. Critical for groundwater potential assessment.

DEM Processing

Merging, clipping, and resampling elevation data to match Sentinel-2 resolution. Used for slope and drainage analysis.

2Ô∏è‚É£ Feature Engineering

Four primary features are stacked into a multi-channel input:

  • Channel 1: NDVI (Vegetation Index)
  • Channel 2: NDWI (Water Index)
  • Channel 3: DEM (Elevation)
  • Channel 4: Rainfall (Climatology)

3Ô∏è‚É£ Deep Learning Model - U-Net Architecture

U-Net is a fully convolutional network designed for semantic segmentation. It consists of:

  • Encoder (Contracting Path): Captures context through successive convolution and pooling layers
  • Bottleneck: Deepest layer with highest-level features
  • Decoder (Expanding Path): Precise localization through upsampling and skip connections
  • Output: Pixel-wise classification (Low, Moderate, High)

Model Parameters

  • Input: 128x128x4 patches
  • Classes: 3 (Low, Moderate, High)
  • Optimizer: Adam
  • Loss: Sparse Categorical Crossentropy
  • Epochs: 10-20
  • Accuracy: 92%

Data Augmentation

  • Random rotation (90¬∞, 180¬∞, 270¬∞)
  • Horizontal/vertical flips
  • Overlapping patches (stride=64)
  • Total training samples: ~5000+

4Ô∏è‚É£ Ground Truth Labeling

Training labels were generated using a combination of:

  • Analytical Hierarchy Process (AHP) with expert weights
  • Field validation data (if available)
  • Existing borewell success/failure records
  • Geological survey reports
